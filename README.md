# MaterialDesignDemo
为了以后开发过程中避免控件和组件的设计问题，特意遵循[Material Design](https://material.google.com/)做一个Demo。Demo包括一些常用控件和组件的UI原型，以后开发可以直接套用。  

# 目录
1. ListView
2. Button
3. CardView
4. Dialog
5. GridView
6. Menu
7. SwipeRecycler
8. Selection
9. Tab
